# LINGNOW 7.2: UNIVERSAL UI/UX INDUSTRIALIZATION STANDARDS

This document defines the **Universal Interaction Loops** and **Architectural Guardrails** for ALL industry archetypes
generated by LingNow.

## 1. The "Loop Guarantee" (No Dead-Ends)

Every prototype MUST implement at least 4 core interaction loops specific to its archetype.

| Archetype     | CORE LOOP 1 (Entry)  | CORE LOOP 2 (Action) | CORE LOOP 3 (Feedback) |
|:--------------|:---------------------|:---------------------|:-----------------------|
| **COMMUNITY** | Feed -> Detail Modal | Like/Comment/Post    | State Toggle / Refresh |
| **DASHBOARD** | Row Click -> Drawer  | Edit/Submit Form     | Toast / Data Reload    |
| **COMMERCE**  | Grid -> Quick View   | Add to Cart          | Drawer Update          |
| **READER**    | TOC -> Content View  | Save Progress        | Success Indication     |
| **LANDING**   | CTA -> Modal Form    | Lead Capture         | Success Message        |

## 2. Navigation & IA (Universal Rules)

### Hierarchy Separation

- **PRIMARY**: Navigation that defines the experience (Sidebar or Top Nav).
- **UTILITY**: Secondary actions (Auth, Search, Notifications). Must be in the Header.
- **PERSONAL**: Account-linked items. Must be in a Dropdown/Widget.
- **OVERLAY**: All details and forms. Must be Modals or Drawers.

### Rule: Z-Index Strategy

- Prototypical depth MUST be managed via a Global State Machine (Alpine.js).
- Avoid page jumps for secondary or tertiary content to maintain browsing momentum.

## 3. High-Fidelity Synthesis (The DNA)

- **Icons**: NO raw SVGs. All icons MUST have `w-x h-x` Tailwind classes.
- **Charset**: Explicit UTF-8 meta tag is MANDATORY.
- **CDNs**: Standard base (Tailwind, Alpine, FontAwesome, Google Fonts) is non-negotiable.

## 4. Engineering Process

- Standard shell: `src/main/resources/templates/StandardShell.html`.
- All agents MUST cross-reference this Skill before outputting code.
